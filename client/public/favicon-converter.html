Favicon Converter Guide

To convert the SVG favicon to ICO format (which some browsers need), follow these steps:

Option 1: Online Converter (Easy)

  1. Go to https://convertio.co/svg-ico/ or any other SVG to ICO converter
  2. Upload the favicon.svg file
  3. Download the converted favicon.ico file
  4. Place the favicon.ico file in this directory (client/public/)

Option 2: Using Command Line Tools

  1. Install Inkscape and ImageMagick if you don't have them
  2. Convert SVG to PNG: inkscape favicon.svg -o favicon.png -w 32 -h 32
  3. Convert PNG to ICO: convert favicon.png favicon.ico

Implementation in HTML

After you have both files, make sure they're referenced in your HTML:

<link rel="icon" type="image/svg+xml" href="/favicon.svg" />
<link rel="alternate icon" type="image/x-icon" href="/favicon.ico" />

The index.html file already has the SVG reference. You may want to add the ICO reference for better browser compatibility.
